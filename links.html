Open Source Platforms & Tools

Build your own: LLM characters, real-time performance systems, and interactive art.

LLM Character Engines

  • RealChar - Open-source realtime AI character platform. Multi-platform (web/mobile/terminal), customizable personality, voice clone support
  • OpenRP - AI roleplaying with world building. GitHub-like collaboration, interconnected characters and lore, custom prompts
  • NPC-Engine - Deep learning toolkit for NPC AI. Real-time dialogue, semantic similarity, TTS integration, engine agnostic via ZMQ/HTTP
  • Long Memory Character Chat - Multi-tiered memory system (short/mid/long-term), vector database, dynamic personality updates
  • Sentient Core - Terminal UI for AI characters. GGUF/Llama.cpp support, GPU acceleration, multi-chat mode, dataset export
  • LLM Interactive Drama Toolkit - Open-source research toolkit from arXiv for LLM-based interactive drama
  • Interactive LLM-Powered NPCs - Voice/webcam integration, facial recognition, SadTalker lip sync, works with existing games (Cyberpunk, GTA)

Real-Time Performance & Interactive Control

  • PraxisLIVE - Hybrid visual live programming. Fork components at runtime, distributed architecture, real-time code reload
  • Synesthesia - Live music visualizer. GLSL shader community, built-in live coding, in-app marketplace for scenes
  • Luz - Fullscreen interactive motion graphics. Wiimote/MIDI/OSC support, unlimited effects layering, live performance tool
  • Awesome Livecoding - Comprehensive list of livecoding tools. SuperCollider, ChucK, FoxDot, Sonic Pi, TidalCycles, and dozens more
  • Open Sound Control (OSC) - Protocol for multimedia device control. Industry standard for connecting audio, visuals, lighting, sensors
  • Notch - Real-time 3D rendering for live events. IMAG integration, DMX/ArtNet/MIDI/OSC control, NotchLC codec (commercial)

Creative Coding Frameworks

  • Processing - Java-based visual arts programming. Extensive libraries, huge community, 2D/3D graphics
  • p5.js - JavaScript version of Processing for the web. Created by Lauren McCarthy, browser-based creative coding
  • openFrameworks - C++ toolkit for creative coding. High performance, extensive multimedia libraries, real-time applications
  • TouchDesigner - Node-based visual programming. Real-time rendering, OSC/MIDI integration, industry standard for installations
  • Awesome Creative Coding - Curated list of creative coding resources, tools, frameworks, and learning materials
